# 太极图形课S1-hw1-双摆(Double Pendulum)
利用taichi的并行特性实现了多个双摆同时运动

## 背景简介
> 双摆是将一根单摆连接在另一个单摆的尾部所构成的系统。双摆同时拥有着简单的构造和复杂的行为。高能量双摆的摆动轨迹表现出对于初始状态的极端敏感。
